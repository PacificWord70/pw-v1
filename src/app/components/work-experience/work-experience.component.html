Technology Development Program

Software Engineering Intern

Over the 2020 summer, I worked as a Virtual Software Engineering Intern at Capital One. I helped build FortDocs, the core document repository for over 45 million documents within their Small Business Technology department. I worked with another intern to onboard the Credit Bureau Dispute team to this platform for upload, search, and edit capabilities. FortDocs is a web-based platform and is hosted on an AWS EC2 instance. The data is stored with an Alfresco Database backend that utilizes OpenCMIS and Chassis Microservices to allow for more utility. For our project, we primarily worked with our Java SpringBoot API that interfaced with the Angular UI made of Typescript and Javascript. I had a great time working with this team and learning more about web-development including entitlements, catalogs, and the Angular framework. I'm grateful to have enjoyed my summer and experience the virtual work environment in these changing times. From this summer I gained remote working skills in communication and productivity as well as technical skills in development.
