Projects

Movie Love

Movie Love is a simple webpage I created that a user can use to create a list of their favorite movies. The user can enter the movie title and year. The list comes with some button features: Star, Remove, and IMDB link.

Upgrade Indy

Upgrade Indy is a project my team and I created during Indy Civic Hacks 2017. This site shows how businesses can elevate the community around them. Indy is a growing city and gaining many new companies and this can really help the citizens. We gathered data from the OpenIndy data portal about land zoning and crime. We allow businesses to view where similar companies to their's are located. I worked in the visualizatitons of the data comprized of the graph and dynamic map.

NoteHerder

NoteHerder is a note keeping web application I created with React JS. It includes authentication with either a Github account or email address. Notes are private to each user and are available through the account used to create them.

Down Write Smart

DWS was created during Boilermake 2017 by my team and I. It is a web tool aimed at elementary school teachers to encourage their students to write more. Each of us struggled with confidence in our writing skills from lack of practice. To counteract this, our project works to encourage these writing skills in younger students through positive re-enforcement and competition. I worked on half of the front-end and helped troubleshoot the back-end.

TechPoint Internship: Xtern Bootcamp
